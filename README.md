# # Blog AI – AI Blog Generator

# Blog AI is a modern web application that generates blog content using AI.
It allows users to quickly create structured blog posts based on a topic or title input.

## Features

* AI-powered blog generation
* Clean and responsive UI
* Fast frontend built with Vite and React
* Supabase backend integration
* Modern component system using shadcn-ui
* Tailwind CSS styling

## Tech Stack

Frontend

* React
* TypeScript
* Vite
* Tailwind CSS
* shadcn-ui

Backend

* Supabase
* Edge Functions

## Installation

1. Clone the repository

```
git clone https://github.com/YOUR_USERNAME/ai-insight-blog-generator.git
```

2. Navigate to the project folder

```
cd ai-insight-blog-generator
```

3. Install dependencies

```
npm install
```

4. Start the development server

```
npm run dev
```

The application will run at:

```
http://localhost:5173
```

## Environment Variables

Create a `.env` file in the root directory and add:

```
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```
